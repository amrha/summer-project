Summer Project

This is a website developed by Amrou Hajji and Med Naser Sayahi as a summer project. The goal is to create a website that indexes images by their content. The user submits a picture then the website outputs the images most likely to be similar to the user's input from a database. This subject was suggested by ENSI teacher Med Amine Mezghich. It was developed using python, the website uses flask and the object recognition uses keras's mobinet.
